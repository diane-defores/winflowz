## Reproduction

- Preconditions:
  - Android real device.
  - Installed debug APK reports `build: sha=37116dd | run=26091472372 | ref=master`.
  - Settings shows `On-device Speech` with French, Hindi, and English entries.
- Steps:
  1. Open WinFlowz Settings.
  2. Go to `On-device Speech`.
  3. Tap `Supprimer le langage pack Hindi`.
  4. Observe whether the UI changes, gives feedback, or updates diagnostics.

## Expected Behavior

- If the Hindi pack is installed or queued, remove should transition it to `removed` with visible feedback.
- If it is already removed/not installed, the UI should not present a misleading remove action, or should show a clear no-op message.
- The diagnostics should make the resulting state clear.
