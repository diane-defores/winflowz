> You probably spend more time in your browser than in any other app. You might as well make it a serious tool.

## Choosing the right browser

Your default browser is not necessarily the one that fits you best. Here are the three most interesting options for an advanced user.

### Vivaldi: the Swiss Army knife

Vivaldi is the power user's choice. It is based on Chromium (so it supports all Chrome extensions) but pushes customization farther than almost any other browser.

**What makes it unique:**

- **Tab Stacking**: stack tabs into visual groups. You can even tile them to view multiple pages at once.
- **Quick Commands** (F2): a universal command bar to search tabs, history, bookmarks, and settings.
- **Command Chains**: automate sequences of actions (open 5 sites, arrange them in a grid, enable reading mode).
- **Side panels**: show notes, calendar, mail, or a web page in a permanent panel.
- **Visual customization**: themes, toolbar layout, shortcuts - everything is configurable.

### Arc: the minimalist approach

Arc rethinks the browser interface. The sidebar replaces the traditional tab bar, spaces separate your contexts (work, personal, projects), and ephemeral tabs disappear automatically.

**Ideal for**: people who want a clean browser and are willing to change old habits.

### Brave: privacy first

Brave blocks ads and trackers by default, without extensions. It is Chromium with built-in protection. Fast, privacy-friendly, and compatible with the Chrome ecosystem.

### Comparison table

| Criterion | Vivaldi | Arc | Brave |
|----------|---------|-----|-------|
| **Base** | Chromium | Chromium | Chromium |
| **Customization** | Extreme | Moderate | Limited |
| **Privacy** | Good | Good | Excellent |
| **Tab management** | Best-in-class | Innovative | Standard |
| **Learning curve** | Medium | High | Low |
| **Best for** | Power users | Minimalists | Privacy-conscious users |

## Managing tabs efficiently

Having 47 tabs open is not multitasking - it is chaos. Here is how to take back control.

### Tab management strategies

- **The 10-tab rule**: if you go beyond 10, it is time to clean up. Bookmark what you want to keep, close the rest.
- **Thematic groups**: create groups by project or context (research, communication, dev).
- **Sessions**: save sets of tabs so you can return to them later without keeping them open.
- **Useful extensions**: OneTab (saves all tabs with one click), Tab Wrangler (automatically closes inactive tabs).
- **Manganum** - a Chrome extension that adds a sidebar with Gmail, Calendar, Drive, Notes, Tasks, and ChatGPT on every tab. Event notifications, built-in translation, and fast access without changing context.

## Essential keyboard shortcuts

These shortcuts work in all Chromium browsers:

| Action | Shortcut |
|--------|----------|
| New tab | `Ctrl + T` |
| Close current tab | `Ctrl + W` |
| Reopen last closed tab | `Ctrl + Shift + T` |
| Next / previous tab | `Ctrl + Tab` / `Ctrl + Shift + Tab` |
| Go to tab N | `Ctrl + 1` to `Ctrl + 8` |
| Last tab | `Ctrl + 9` |
| Address bar | `Ctrl + L` or `F6` |
| Find on page | `Ctrl + F` |
| Open link in new tab | `Ctrl + Click` |
| Private browsing | `Ctrl + Shift + N` |

**Tip**: `Ctrl + Shift + T` is probably the most underrated shortcut. It can reopen several tabs in a row.

## Installing extensions from outside the store

Sometimes the extension you need is not on the Chrome Web Store. To install a `.crx` file or an unpacked folder:

1. Open `chrome://extensions/` (or the equivalent in your browser)
2. Enable **Developer mode** in the top-right corner
3. Click **Load unpacked** and select the folder
4. The extension appears in your list

**Warning**: only install extensions from sources you trust. A malicious extension has access to everything you do in your browser.

## Optimizing performance

- **Disable unused extensions**: every extension consumes memory
- **Limit the number of tabs**: each tab is a separate process
- **Enable hardware acceleration**: `Settings > System > Hardware acceleration`
- **Clear the cache regularly**: `Ctrl + Shift + Delete`
