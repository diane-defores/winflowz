Keyboard shortcuts are the **silent superpower** of every efficient user. Every second saved adds up: over the course of a year, you reclaim hours of work. This module gives you the shortcuts you should know by heart.

## Essential Windows shortcuts

| Shortcut | Action |
|----------|--------|
| `Win + Tab` | Task View (virtual desktops) |
| `Win + Ctrl + ←/→` | Switch between virtual desktops |
| `Win + Ctrl + D` | Create a new virtual desktop |
| `Win + Ctrl + F4` | Close the active virtual desktop |
| `Win + ←/→/↑/↓` | Snap a window into place |
| `Win + I` | Open Settings |
| `Win + X` | Power User menu |
| `Win + R` | Run command |
| `Win + .` | Open emoji picker |
| `Win + A` | Notification center |
| `Win + P` | Projection options (displays) |
| `Alt + Tab` | Switch between windows |

### Text manipulation

| Shortcut | Action |
|----------|--------|
| `Ctrl + A` | Select all |
| `Ctrl + D` | Duplicate |
| `Ctrl + Z` / `Ctrl + Y` | Undo / Redo |
| `Ctrl + B` / `I` / `U` | Bold / Italic / Underline |
| `Ctrl + ←/→` | Move word by word |
| `Ctrl + Shift + ←/→` | Select word by word |
| `Ctrl + Delete` | Delete an entire word |

:::tip[Pro tip]
You do not need to release `Ctrl` between consecutive shortcuts. For example, to select all and then copy: hold `Ctrl`, press `A`, then `C`. Smooth and fast.
:::

## Learn shortcuts effectively

The key is not to memorize everything at once, but to **add 2-3 shortcuts per week** into your routine. Here's the method:

1. **Identify** an action you do often with the mouse
2. **Find** the matching shortcut
3. **Force yourself** to use it for 3 days, even if it feels slower at first
4. **Move on** to the next shortcut once it becomes automatic

## Going further with AutoHotkey

If a shortcut does not exist, **create it yourself**. [AutoHotkey](https://www.autohotkey.com/) is a free Windows tool that lets you program your own shortcuts. A few ideas:

- **Remap unused keys** (for example, `Caps Lock` -> `Esc`)
- **Create text expansions** (for example, typing `@@` -> your email address)
- **Automate click and keystroke sequences** for repetitive tasks

The philosophy behind cheatsheets is simple: **do not memorize everything, know where to look**. Keep your references close at hand and save your muscle memory for the 20 shortcuts you actually use most often.
