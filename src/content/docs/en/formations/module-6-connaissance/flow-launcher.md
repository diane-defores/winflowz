[Flow Launcher](https://www.flowlauncher.com/) is the open-source productivity launcher for Windows that replaces Start menu search, the file manager, and a dozen daily clicks.

> Once you get used to a keyboard launcher, going back to the mouse just to open an app feels as archaic as a 56k modem.

## What is Flow Launcher?

Flow Launcher is a keyboard-based application launcher: you press a shortcut, type a few letters, press Enter. Done.

### What it does

| Function | Example |
|----------|---------|
| **Launch an app** | Type `fire` -> opens Firefox |
| **Find a file** | Type `> quarterly report` -> finds the file |
| **Calculate** | Type `15 * 1.2` -> shows the result |
| **Search the web** | Type `g astro starlight` -> Google search |
| **Open a bookmark** | Type `b project` -> opens the Vivaldi bookmark |
| **System commands** | Type `shutdown`, `restart`, `sleep` |

## Installation and setup

### Installation

1. Download it from [flow-launcher.com](https://www.flowlauncher.com/)
2. Install and launch - Flow Launcher starts with Windows
3. The default shortcut is `Alt + Space`

### Recommended settings

- **Shortcut**: keep `Alt + Space` (universal and ergonomic)
- **Theme**: choose a clean dark theme - you will see it 50 times a day
- **Number of results**: 5-6 max so the list stays focused
- **Position**: centered at the top of the screen (the natural line of sight)

## Integration with Vivaldi bookmarks

One of [Flow Launcher](https://www.flowlauncher.com/)'s superpowers: access your Vivaldi bookmarks directly from the launcher.

### Setup

1. Install the **Bookmarks** plugin in Flow Launcher (Settings -> Plugin Store)
2. Point it to the [Vivaldi](https://vivaldi.com/) bookmarks file:
   `%LOCALAPPDATA%\\Vivaldi\\User Data\\Default\\Bookmarks`
3. Set an action prefix: `b` (for bookmarks)

### Usage

- `b project` -> shows all bookmarks containing "project"
- `b github winflowz` -> opens the repo directly
- No need to open the browser just to find a bookmark

## Essential shortcuts and plugins

### Recommended plugins

| Plugin | Prefix | Function |
|--------|--------|----------|
| [Everything](https://www.voidtools.com/) | none | Ultra-fast file search |
| **Bookmarks** | `b` | Access browser bookmarks |
| **Calculator** | none | Direct calculations in the bar |
| **Shell** | `>` | Run terminal commands |
| **Clipboard History** | `cb` | Clipboard history |
| **Colors** | `#` | Color preview and conversion |

### Keyboard shortcuts

| Shortcut | Action |
|-----------|--------|
| `Alt + Space` | Open/close Flow Launcher |
| `Tab` | Autocomplete |
| `Enter` | Open the first result |
| `Ctrl + Enter` | Open the parent folder of the file |
| `Shift + Enter` | Run as administrator |

## Why a launcher changes everything

### The math

- **50 actions/day** where you search for an app, file, or bookmark
- **Without a launcher**: 5-10 seconds per action (Start menu, browser, file explorer)
- **With a launcher**: 1-2 seconds per action
- **Gain**: around 5-7 minutes per day, 25-35 minutes per week

But the real gain is not time - it is **flow**. You never leave the keyboard, and you never lose concentration looking for a tool. Your intention turns directly into action.

### Before / After

| Before | After |
|-------|-------|
| Click Start menu -> search -> click | `Alt+Space` -> 3 letters -> `Enter` |
| Open browser -> bookmarks -> folder -> click | `Alt+Space` -> `b keyword` -> `Enter` |
| Open file explorer -> navigate -> search | `Alt+Space` -> `> file-name` -> `Enter` |
| Open calculator -> type -> read | `Alt+Space` -> `15*1.2` -> read the result |

## Advanced tips

1. **Customize keywords** - rename your frequent apps with short aliases
2. **Disable unused plugins** - less noise in the results
3. **Use web shortcuts** - `yt [term]` for YouTube, `gh [repo]` for GitHub
4. **Pin frequent results** - they will appear first
5. **Combine it with [Everything](https://www.voidtools.com/)** - Flow Launcher + Everything = total instant search
