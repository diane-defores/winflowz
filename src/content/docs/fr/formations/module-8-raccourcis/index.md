Les raccourcis clavier, c'est le **superpower silencieux** de tout utilisateur efficace. Chaque seconde gagnée s'accumule : sur une année, tu récupères littéralement des heures de travail. Ce module te donne les raccourcis essentiels à connaître par cœur.

## Raccourcis Windows indispensables

| Raccourci | Action |
|-----------|--------|
| `Win + Tab` | Vue des tâches (bureaux virtuels) |
| `Win + Ctrl + ←/→` | Basculer entre les bureaux virtuels |
| `Win + Ctrl + D` | Créer un nouveau bureau virtuel |
| `Win + Ctrl + F4` | Fermer le bureau virtuel actif |
| `Win + ←/→/↑/↓` | Ancrer une fenêtre (snap) |
| `Win + I` | Ouvrir les Paramètres |
| `Win + X` | Menu d'accès rapide (Power User) |
| `Win + R` | Exécuter une commande |
| `Win + .` | Ouvrir le sélecteur d'emojis |
| `Win + A` | Centre de notifications |
| `Win + P` | Options de projection (écrans) |
| `Alt + Tab` | Basculer entre les fenêtres |

### Manipulation de texte

| Raccourci | Action |
|-----------|--------|
| `Ctrl + A` | Tout sélectionner |
| `Ctrl + D` | Dupliquer |
| `Ctrl + Z` / `Ctrl + Y` | Annuler / Rétablir |
| `Ctrl + B` / `I` / `U` | Gras / Italique / Souligné |
| `Ctrl + ←/→` | Se déplacer mot par mot |
| `Ctrl + Maj + ←/→` | Sélectionner mot par mot |
| `Ctrl + Suppr` | Supprimer un mot entier |

:::tip[Astuce pro]
Tu n'as pas besoin de relâcher `Ctrl` entre deux raccourcis enchaînés. Par exemple, pour tout sélectionner puis copier : maintiens `Ctrl`, tape `A` puis `C`. Fluide et rapide.
:::

## Apprendre ses raccourcis efficacement

La clé n'est pas de tout mémoriser d'un coup, mais d'**intégrer 2-3 raccourcis par semaine** dans ta routine. Voici la méthode :

1. **Identifie** une action que tu fais souvent à la souris
2. **Cherche** le raccourci correspondant
3. **Force-toi** à l'utiliser pendant 3 jours — même si c'est plus lent au début
4. **Passe** au raccourci suivant une fois que c'est automatique

:::note[Ressources cheatsheets]
- [UseTheKeyboard](https://usethekeyboard.com/) — raccourcis pour des centaines d'apps (Mac et Windows)
- [OverAPI.com](https://overapi.com/) — cheatsheets pour langages et outils de dev
- [Devhints.io](https://devhints.io/) — résumés concis pour développeurs
:::

## Aller plus loin avec AutoHotkey

Si un raccourci n'existe pas, **crée-le toi-même**. [AutoHotkey](https://www.autohotkey.com/) est un outil gratuit pour Windows qui te permet de programmer tes propres raccourcis. Quelques idées :

- **Remapper des touches** inutilisées (ex. `Verr Maj` → `Échap`)
- **Créer des abréviations** qui s'étendent en texte complet (ex. taper `@@` → ton adresse email)
- **Automatiser des séquences** de clics et de frappes pour les tâches répétitives

La philosophie derrière les cheatsheets est simple : **ne mémorise pas tout, sache où chercher**. Garde tes références à portée de main, et concentre ta mémoire musculaire sur les 20 raccourcis que tu utilises le plus souvent.
